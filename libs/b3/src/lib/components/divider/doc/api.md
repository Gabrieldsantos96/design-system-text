# API

## [b3-divider] <span class="api-type-label component">Component</span>

> `b3-divider` is used to visually separate content with a horizontal or vertical line.

| Property       | Description                   | Type                         | Default      |
| -------------- | ----------------------------- | ---------------------------- | ------------ |
| `zOrientation` | Defines the divider direction | `"horizontal" \| "vertical"` | `horizontal` |

## March 2025 - Foundation Release

March marked the foundation of B3UI with the first components and testing infrastructure.

## New Components

#### ✨Button Component

The **Button component** was the first component created for B3UI, establishing the foundation of the design system.

```angular-html
<button b3-button>Default</button>
<button b3-button zType="secondary">Secondary</button>
<button b3-button zType="destructive">Destructive</button>
```

Features include multiple variants, loading states, and accessibility support.

#### ✨Card Component

Created the **Card component** with documentation and examples.

```angular-html
<b3-card>
  <b3-card-header>Card Title</b3-card-header>
  <b3-card-body>Card content</b3-card-body>
</b3-card>
```

Enhanced with header and body components for better structure.

#### ✨Checkbox Component

Implemented a **Checkbox component** with disabled state support and improved accessibility.

```angular-html
<b3-checkbox>Accept terms</b3-checkbox>
<b3-checkbox disabled>Disabled option</b3-checkbox>
```

#### ✨Input Components

Created **Input and Textarea components** for form inputs.

```angular-html
<b3-input placeholder="Enter text"></b3-input>
<b3-textarea placeholder="Enter description"></b3-textarea>
```

#### ✨Badge Component

Added a **Badge component** with comprehensive documentation and demos.

```angular-html
<b3-badge>Active</b3-badge>
<b3-badge variant="secondary">Secondary</b3-badge>
```

#### ✨Switch Component

Added a **Switch component** for binary toggle controls.

```angular-html
<b3-switch>Enable notifications</b3-switch>
```

#### ✨Radio Component

Implemented a **Radio component** for single-choice selections.

```angular-html
<b3-radio value="option1">Option 1</b3-radio>
<b3-radio value="option2">Option 2</b3-radio>
```

#### ✨Loader Component

Created a versatile **Loader component** for indicating loading states.

```angular-html
<b3-loader></b3-loader>
```

#### ✨Tooltip Component

Introduced a **Tooltip component** with smart positioning and lifecycle management.

```angular-html
<button b3-button>
  Hover me
  <!-- Tooltip content -->
</button>
```

#### ✨Divider Component

Added a **Divider component** for content separation.

```angular-html
<b3-divider></b3-divider>
<b3-divider orientation="vertical"></b3-divider>
```

#### ✨Alert Component

Created a comprehensive **Alert system** for user notifications.

```angular-html
<b3-alert zType="success">
  <b3-alert-icon></b3-alert-icon>
  <b3-alert-title>Success!</b3-alert-title>
  <b3-alert-description>Your changes have been saved.</b3-alert-description>
</b3-alert>
```

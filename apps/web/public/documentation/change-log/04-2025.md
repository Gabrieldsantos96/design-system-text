#### Dropdown Component

Added a powerful **Dropdown component** for contextual menus.

```angular-html
<b3-dropdown>
  <b3-dropdown-trigger>Menu</b3-dropdown-trigger>
  <b3-dropdown-content>
    <b3-dropdown-item>Edit</b3-dropdown-item>
    <b3-dropdown-item>Delete</b3-dropdown-item>
    <b3-dropdown-divider></b3-dropdown-divider>
    <b3-dropdown-item>Settings</b3-dropdown-item>
  </b3-dropdown-content>
</b3-dropdown>
```

Features include smart positioning, nested menus support, and trigger event control.

#### Avatar Component

Implemented a flexible **Avatar component** for user representation.

```angular-html
<b3-avatar>
  <b3-avatar-image src="/user.jpg" alt="User Name"></b3-avatar-image>
  <b3-avatar-fallback>UN</b3-avatar-fallback>
</b3-avatar>
```

Features include image, initials, or icon fallback, multiple sizes, and status indicators.

#### Tabs Component

Created a versatile **Tabs system** for organizing content.

```angular-html
<b3-tabs defaultValue="overview">
  <b3-tabs-list>
    <b3-tabs-trigger value="overview">Overview</b3-tabs-trigger>
    <b3-tabs-trigger value="details">Details</b3-tabs-trigger>
  </b3-tabs-list>
  <b3-tabs-content value="overview">Overview content</b3-tabs-content>
  <b3-tabs-content value="details">Details content</b3-tabs-content>
</b3-tabs>
```

Features include horizontal and vertical orientation, lazy loading, and animated transitions.

#### Select Component

Developed an advanced **Select dropdown** with enhanced functionality.

```angular-html
<b3-select>
  <b3-select-trigger>
    <b3-select-value placeholder="Choose an option..."></b3-select-value>
  </b3-select-trigger>
  <b3-select-content>
    <b3-select-item value="option1">Option 1</b3-select-item>
    <b3-select-item value="option2">Option 2</b3-select-item>
  </b3-select-content>
</b3-select>
```

Features include built-in filtering, multiple selection support, and performance optimization.

#### Accordion Component

Finally added the **Accordion component**, which became the new default component (replacing Alert).

```angular-html
<b3-accordion>
  <b3-accordion-item>
    <b3-accordion-trigger>Getting Started</b3-accordion-trigger>
    <b3-accordion-content>
      Quick setup instructions and basic usage examples.
    </b3-accordion-content>
  </b3-accordion-item>
</b3-accordion>
```

Features include single or multiple expansion control, smooth CSS transitions, and full keyboard accessibility. The Accordion component availability was updated to true and set as the default component.
